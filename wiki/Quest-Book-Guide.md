# Quest Book Guide

## Layout

The quest book is split into:

- Category/Sub-category navigation
- Quest list
- Quest details panel

## Browsing and Search

- Use categories to narrow scope quickly.
- Use search when you know part of the quest name.
- Switch filters to show/hide completed, rejected, or locked quests (based on settings).

## Reading Quest Details

The details panel shows:

- Quest name and icon
- Description
- Requirements/dependencies
- Objective progress
- Reward preview

## Dependency Behavior

Some quests stay locked until prerequisites are completed.  
If a pack uses dependency lock behavior, progression is strictly ordered.

## Rejection and Repeat

- **Reject** is only available for optional quests.
- **Repeat** appears on repeatable quests after completion.

## Completion Scroll Flow

If enabled by settings/pack configuration, quest scroll actions can be used directly from the quest UI.

## Pinned Quests

Pinning lets you keep selected quest progress visible in the HUD while playing.
