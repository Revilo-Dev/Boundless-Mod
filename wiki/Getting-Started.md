# Getting Started

## 1) Install and Launch

Install Boundless like any other NeoForge mod, then start the game.

## 2) Get the Quest Book

You can access quests through:

- The `Quest Book` item
- The keybind (`[` by default)
- The inventory quest button (if enabled in settings)

## 3) Open the Quest UI

When you open the quest book, you can:

- Browse by category/sub-category
- Search quest names
- Open quest details
- Track objective progress

## 4) Complete and Redeem Quests

Depending on the quest, you may need to:

- Hold required items
- Submit items
- Kill entities
- Gain effects
- Trigger advancements
- Reach stat/xp targets
- Enter text input

When a quest is ready, use the claim/complete action in the quest UI.

## 5) Optional Systems

Based on settings and pack design, you may also have:

- Quest rejection (optional quests only)
- Repeat flow for repeatable quests
- Pinned quest HUD
- Quest completion scroll support

## Common First-Time Checks

- If the quest button is missing, check settings.
- If a quest does not progress, confirm the objective type and exact target id.
- If you edited pack files manually, run `/boundless reload`.
