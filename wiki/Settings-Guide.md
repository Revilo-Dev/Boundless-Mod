# Settings Guide

Boundless includes an in-game settings screen with grouped options for UI, functionality, and gameplay behavior.

## What You Can Control

Typical options include:

- Questbook visibility/access behavior
- Search box visibility behavior
- Filter display behavior
- Category visibility behavior
- Pinning and HUD behavior
- Toast notification behavior
- Auto-claim/auto-complete related behavior
- Quest scroll behavior

## Suggested Baselines

### Survival Servers

- Keep notifications on (toasts + HUD pinning optional)
- Keep filters available
- Disable high-noise options if players report UI clutter

### Singleplayer Progression

- Enable pinning
- Keep search and filters enabled
- Use auto-claim only if you want faster flow between quest chains

## If a Setting Seems Ignored

- Check if the feature is also constrained by quest data.
- Reopen the quest screen after changing options.
- On multiplayer, confirm server-side behavior is expected for that feature.
